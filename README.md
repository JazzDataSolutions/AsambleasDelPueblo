# Asambleas del Pueblo

Plataforma de democracia participativa. Organización, documentos, herramientas y contenidos para la construcción del poder popular desde la base.

## Stack

- **Next.js 16** (App Router, SSG)
- **TypeScript**
- **Tailwind CSS v4**

## Development

```bash
npm install
npm run dev
```

## Secciones

- Contacto (correo, WhatsApp, YouTube, Facebook, Instagram, TikTok)
- Documentos (Manifiesto, APS Escrito, Metodología, Materiales de Consulta)
- Justificación Legal (Herramientas Jurídicas)
- Diagnóstico Territorial (Problemas Locales, Asamblea como atención)
- Herramientas Digitales (Consulta, Encuesta, Iniciativas, Encuentros)
- Solicitud de Equipos (Celular, Micrófonos, Computadora, Proyector)
- Contenidos (Entrevistas, Análisis, Historia, Democracia Participativa)
